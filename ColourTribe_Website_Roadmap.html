Tech Stack & Project Setup

Every library chosen for a specific reason — performance, animation quality, or DX. No over-engineering.

⚡

React 18 + Vite

Lightning fast HMR during dev. SWC transpiler. Tree-shaking for production. Code-splitting per route via React.lazy().

🎨

Tailwind CSS v3

Utility-first CSS with custom config for brand colors (navy, gold, cream). JIT mode. No unused CSS in prod.

🎭

Framer Motion v11

All entrance animations, scroll reveals, page transitions, hover states, and the homepage particle / cloth simulations.

🔥

Firebase v10

Firestore for data, Storage for images, Auth for admin login, Hosting for deployment, Functions for quote emails.

🗺️

React Router v6

File-based routing pattern. Nested routes for admin. Lazy-loaded pages. Scroll restoration on route change.

📦

Zustand

Lightweight global state for quote cart, lead info, UI state. No Redux overhead. Persisted to localStorage.

📋

React Hook Form + Zod

All forms (lead modal, quote request, contact). Type-safe validation with Zod schemas. No controlled-input re-renders.

🖼️

Three.js (R3F)

For the homepage 3D rotating fabric/cloth mesh background effect. React Three Fiber wrapper for clean integration.

Firestore Security Rules

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {

    // Anyone can READ products and categories
    match /products/{id} {
      allow read: if true;
      allow write: if request.auth != null && isAdmin();
    }
    match /categories/{id} {
      allow read: if true;
      allow write: if request.auth != null && isAdmin();
    }

    // Anyone can CREATE leads/quotes (no auth needed)
    match /leads/{id} {
      allow create: if true;
      allow read, update, delete: if request.auth != null && isAdmin();
    }
    match /quotes/{id} {
      allow create: if true;
      allow read, update, delete: if request.auth != null && isAdmin();
    }

    // Only admin reads settings
    match /settings/{id} {
      allow read: if true;
      allow write: if request.auth != null && isAdmin();
    }

    function isAdmin() {
      return request.auth.token.admin == true;
    }
  }
}

Lead Capture Entry Modal

The first thing a visitor sees. A visually beautiful, non-intrusive full-screen overlay that captures Name + Phone before they browse. Shows once per session (sessionStorage flag). After submit, data goes to Firestore /leads and the modal closes.

Visual Design Spec

  • Backdrop: Navy (#0f1b2d) with 0.92 opacity blur backdrop. A subtle animated gold particle dust floats behind the modal box (canvas-based, 30 particles).
  • Modal Box: White card, 480px wide, centered, border-radius 16px. Top accent bar: 4px gradient (gold → orange → gold). Subtle drop shadow: 0 25px 60px rgba(0,0,0,0.4).
  • Logo: Colour Tribe logo SVG at top-center, 64px tall, with a slow spin entrance animation (rotateY 0°→360° once, 0.8s).
  • Headline: "Get Bulk Pricing Today" in Playfair Display 28px. Subtext: "Tell us who you are — we'll show you the right prices." 14px Inter light.
  • Fields: Name input + Phone input. Clean floating-label style. Gold bottom border on focus. Phone field auto-prefixes "+91".
  • CTA Button: Full-width, navy background, gold text. Hover: reverses (gold bg, navy text). Ripple click effect via Framer Motion.
  • Skip link: "Browse without registering →" in small muted text below button. Closes modal, does NOT save lead.

Animation Spec

🎭 Framer Motion — Modal Entrance

Backdrop: opacity 0 → 1, duration 0.3s ease.
Modal box: scale(0.85) + y(+40px) → scale(1) + y(0), duration 0.45s, spring(stiffness:300, damping:25).
Fields + button: staggered children fade-up, 0.08s delay each.
Exit: scale(0.9) + opacity 0, duration 0.2s. Backdrop fades out simultaneously.

Logic

// hooks/useLeadModal.js — simplified
const shouldShowModal = () => {
  const seen = sessionStorage.getItem('ct_lead_captured');
  return !seen;  // show once per session
};

const submitLead = async (data) => {
  await addDoc(collection(db, 'leads'), {
    ...data,
    source: 'modal',
    status: 'new',
    createdAt: serverTimestamp()
  });
  sessionStorage.setItem('ct_lead_captured', 'true');
  leadStore.setState({ name: data.name, phone: data.phone });
};

Homepage — Sections & Animations

The homepage is the centrepiece of the visual experience. Every section has a specific animation choreography. Build top-to-bottom in this exact order.

Section 1 — Hero

HeroSection.jsx

Full Viewport Height — 3D Animated Background

Layout

  • Background Layer (z:0): React Three Fiber canvas fills 100vw × 100vh. Scene: A large low-poly cloth/fabric mesh (PlaneGeometry 20×20, 60×60 segments) that undulates like fabric blowing in wind using a vertex shader animating Y position with sin waves. Color: dark navy (#0a1520) with gold (#c9a84c) light reflecting from one top-right point light. The mesh rotates very slowly (0.0003 rad/frame on Y axis).
  • Gradient Overlay (z:1): Position absolute, full coverage. Linear gradient: transparent top 40% → navy bottom 100%. Creates depth so text is readable.
  • Content (z:2): Centered flex column. Left-aligned on md+.
  • Headline: Playfair Display, 68px desktop / 40px mobile. Line 1: "Dress Your" in white. Line 2: "Hospitality" in gold (animated character-by-character). Line 3: "in Style." in white. Total 3 lines staggered.
  • Sub-headline: "Bulk uniform manufacturer for hotels, restaurants, events & corporates across India. Minimum order: 10 pieces." — 18px Inter light, max-width 560px, color rgba(255,255,255,0.72).
  • CTA Row: Two buttons side-by-side. "Browse Catalogue" (solid gold, navy text, arrow icon). "Get a Quote" (outline white, hover fills white).
  • Trust badges: Below CTA — 3 badges in a row: ✦ 500+ Hotel Clients | ✦ In-House Manufacturing | ✦ Delhi NCR & Pan-India Delivery
  • Scroll indicator: Bottom-center. Animated chevron bouncing downward. Disappears after user scrolls 100px.
🎭 Hero Text Animation (Framer Motion)

Each word of the headline is wrapped in a <motion.span>. Initial: {opacity:0, y:40}. Animate: {opacity:1, y:0}. Stagger: 0.12s between words. Duration: 0.6s each. Easing: [0.22, 1, 0.36, 1] (custom cubic bezier). CTA buttons appear after all headline words complete (total ~0.8s delay).

✨ Gold Text Shimmer Effect

"Hospitality" word has CSS animation: a repeating linear-gradient shimmer moving left-to-right every 2.5s. background: linear-gradient(90deg, #c9a84c 0%, #f0d080 50%, #c9a84c 100%). background-size: 200%. animation: shimmer 2.5s linear infinite.

Quote Builder & Cart System

Zustand Quote Store

// store/quoteStore.js
import { create } from 'zustand';
import { persist } from 'zustand/middleware';

export const useQuoteStore = create(
  persist(
    (set, get) => ({
      items: [],  // { productId, sku, name, image, qty, color, sizes, unitPrice, lineTotal }

      addItem: (product, config) => set(state => {
        const existing = state.items.find(i => i.productId === product.id);
        if (existing) return { items: state.items.map(i =>
          i.productId === product.id ? { ...i, qty: i.qty + config.qty } : i
        )};
        return { items: [...state.items, { productId: product.id, ...config }] };
      }),

      updateQty: (productId, qty) => set(state => ({
        items: state.items.map(i => i.productId === productId ? { ...i, qty } : i)
      })),

      removeItem: (productId) => set(state => ({
        items: state.items.filter(i => i.productId !== productId)
      })),

      clearAll: () => set({ items: [] }),

      getApproxTotal: () => get().items.reduce((sum, i) => sum + i.lineTotal, 0),
      getItemCount:   () => get().items.reduce((sum, i) => sum + i.qty, 0),
    }),
    { name: 'ct-quote-cart' }
  )
);

Bulk Price Calculator — priceCalc.js

// utils/priceCalc.js
export function calculateLineTotal(product, qty) {
  const bulkTiers = product.bulkPricing || [];  // sorted ascending by minQty
  let unitPrice = product.basePrice;
  for (const tier of bulkTiers) {
    if (qty >= tier.minQty) unitPrice = tier.price;
  }
  return { unitPrice, lineTotal: unitPrice * qty };
}

// Default bulk tiers (admin can override per product):
// 1–9:    base price (single unit not offered, show warning)
// 10–49:  base × 0.94 (~6% off)
// 50–99:  base × 0.87 (~13% off)
// 100+:   base × 0.80 (~20% off)

Performance Checklist

  • All images as WebP, max 800px width, <120KB each. Use srcset for responsive sizing.
  • Route-based code splitting with React.lazy + Suspense. Each page loads its JS chunk on demand.
  • Google Fonts loaded with font-display: swap and preconnect hints in index.html.
  • Three.js loaded lazily only on homepage. HeroSection wraps canvas in dynamic import.
